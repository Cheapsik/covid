COVID-19 Info App
</h1>


## General info
The project collects data from each country and displays the number of infections, healings and deaths from COVID-19.

	
## Development
The website is built on React.JS
- React: 17.0.2
- SASS: 1.32.8
- Version: 0.1.0
	
## Setup
To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
